# Tokyo Panel - Shopify Theme

A dark, manga-inspired Shopify theme for Tokyo Panel clothing brand.

## Features

- **Dark Manga Aesthetic**: Black background with sleek white glow outlines
- **Responsive Design**: Works seamlessly on all devices
- **Shopify Integration**: Full integration with Shopify/Printify for product management
- **Pages Included**:
  - Home page with hero section
  - Catalogue/Collection page
  - About Us page
  - Contact page
  - Individual product pages

## Theme Structure

```
gbs/
├── assets/          # CSS, JS, and image files
├── config/          # Theme configuration
├── layout/          # Theme layout files
├── sections/        # Reusable sections
├── snippets/        # Reusable code snippets
└── templates/       # Page templates
```

## Installation

1. Upload this theme to your Shopify store
2. Go to Online Store > Themes
3. Click "Add theme" > "Upload zip file"
4. Select the theme files and upload
5. Customize the theme in the theme editor

## Customization

The theme can be customized through the Shopify theme editor:
- Colors (background, text, glow)
- Header logo text
- Footer slogan
- Hero section content
- Featured collections

## Design Philosophy

The theme embraces a dark, manga-inspired aesthetic with:
- Black backgrounds (#000000)
- White glow effects for text and borders
- Manga-style typography
- Sleek, modern design elements
